# 🧠 Quiz Generator App (Full Stack)

A full-stack quiz platform similar to Wayground/Kahoot where users can:
- Create quizzes (MCQ-based)
- Share quiz links
- Attempt quizzes with timer
- View leaderboard & analytics
- Prevent duplicate attempts using email
- Edit & manage quizzes

---

## 🚀 Tech Stack

### Frontend
- React + TypeScript
- React Router
- Axios
- Custom CSS (dark/light theme)

### Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication

---

## 📦 Features

### 👤 User
- Signup / Login
- Dashboard
- Create quizzes
- Edit quizzes
- End quizzes
- View stats

### 🧪 Quiz System
- MCQ-based questions
- Timer per quiz
- Auto submit on timeout
- Email-based attempt restriction
- Score calculation

### 📊 Analytics
- Leaderboard per quiz
- Name + Score + Time taken
- Sorted ranking system
